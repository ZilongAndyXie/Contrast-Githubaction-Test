modify these files to disable the stacktrace leaking whitelabel error page and add messages for a custom error page

resources/application.properties:
server.error.whitelabel.enabled=false

resources/messages_de.properties:
error.error=Fehler
error.generalerrormessage=Etwas ist schiefgelaufen!

resources/messages_en.properties:
error.error=Error
error.generalerrormessage=Something went wrong!

add these files as a catchall error handler and catchall error page

controller/ErrorHandlerController.java
webapp/WEB-INF/error.jsp

reminder: Exceptions are logged automatically by Spring Boot. Ideally, all application exceptions should be catched in their respective controllers or in a global ControllerAdvice ExceptionHandler.
